# HSM Management — Feature Plans

This directory tracks feature plans, both pending and completed, for the HSM Management system.

## Structure

```
plans/
├── README.md           # This file — index of all plans
└── features/           # One file per planned/in-progress feature
    └── teacher360-view.md
```

## Plan Status Key

| Status | Meaning |
|---|---|
| **Draft** | Being designed, not yet approved |
| **Approved** | Reviewed and ready for implementation |
| **In Progress** | Currently being built |
| **Completed** | Shipped to production |
| **Deferred** | Deprioritized, not dropped |
