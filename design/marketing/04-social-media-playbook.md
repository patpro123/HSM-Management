## Instagram Strategy (@hyderabadschoolofmusic)

### Current State
- **133 followers, 170 posts** — strong content volume, weak distribution
- Problem: Static images + inconsistent posting = Instagram deprioritises the account
- Fix: Shift to **Reels** (Instagram's highest-reach format), consistent 3x/week schedule

### 90-Day Growth Target: 133 → 500+ followers

### Content Pillars (3 per week, rotating)

| Pillar | Format | Cadence | Purpose |
|---|---|---|---|
| Student Spotlight | Reel (15–30s) | 1x/week | Social proof + emotional resonance |
| Education / Tips | Carousel (4–6 slides) | 1x/week | Value delivery, saves, shares |
| Behind the Scenes | Reel or Story | 1x/week | Authenticity, teacher personality |

### Reel Formula (Every Reel)

```
[0–2 sec]  Hook: surprising visual or bold text
            e.g. "This 8-year-old has been playing Tabla for 3 months →"

[2–10 sec] Payoff: the student playing / teacher teaching / transformation
            Overlay text: name, instrument, months studied

[10–15 sec] Soft CTA: "Want to start your journey?"
            Screen tap → link in bio → trial booking
```

**Audio:** Always use trending audio from the Instagram audio library. If student performance audio is available and high-quality, use original audio — Instagram boosts original audio.

### Growth Tactics

1. **Reel frequency:** 3 Reels per week for the first 60 days — this is non-negotiable for growth
2. **Collaborations:**
   - DM 3–5 Hyderabad parent/mom influencers (5K–50K followers) for a "free month" exchange
   - Tag local community accounts: @suncity.hyderabad, @kismatpur.life, @hyderabad_parents (or equivalents — search for active local accounts)
3. **Contest (Month 1):**
   - "Share this post + tag a friend who should learn music → win 1 free month"
   - Run for 2 weeks, announce winner via Stories
4. **Comment engagement:** Reply to every comment within 2 hours. Instagram rewards accounts with high engagement velocity.
5. **Cross-post trigger:** Every new Google review → create a Story quoting it (screenshot with branded overlay)
6. **Recital content:** Film 5–10 student performances at next recital. This creates 2 months of Reel content.

## Facebook Strategy

### Goal: Parent community hub + event marketing

### Content Plan
- **Mirror all Instagram posts** (auto-share or manual repost)
- **Long-form posts** (Facebook audiences read more):
  - "Why we started Hyderabad School of Music" — founder story
  - "Our annual recital is coming up — here's what to expect"
  - "5 things we've learned from teaching 100+ students music"
- **Events:** Create a Facebook Event for every recital, open house, and workshop (minimum 2 weeks before). Invite all followers.
- **Facebook Group:** Create "HSM Families" — private group for enrolled students/parents
  - Purpose: recital announcements, schedule changes, student achievements, parent Q&A
  - This builds loyalty and reduces churn

### Facebook Ads (Month 2)
- **Budget:** ₹3,000–5,000/month to start
- **Audience:** Parents aged 28–45, Hyderabad city (10km radius), interests: "children's education", "music", "performing arts"
- **Ad creative:** Best-performing Instagram Reel boosted as Facebook ad
- **CTA:** "Book your child's first free class"
- **Retargeting:** People who visited the website but didn't submit the trial form

## WhatsApp Business Strategy

### Setup
1. Create WhatsApp Business account (separate from personal)
2. Set up business profile: name, address, hours, description, website link
3. Create automated welcome message: "Hi! 👋 Thanks for reaching out to HSM. We'll respond within a few hours. In the meantime, visit our website to see all our instruments: [link]"
4. Set up Quick Replies for FAQs:
   - "Fee": "Classes start from ₹XXXX/month. Your first demo class is completely free — no commitment! Book here: [link]"
   - "Schedule": "We're open Tue–Fri 5–9PM, Sat 3–9PM, Sun 10AM–1PM & 5–9PM. Monday closed."
   - "Demo": "Book your free demo class here: [link] or just reply with your name + preferred instrument 🎵"

## YouTube Strategy (Month 2+)

### Why YouTube
- Videos rank in Google Search (e.g. "learn tabla Hyderabad" → HSM YouTube video)
- Recital content lives permanently and can be shared with families
- Parent testimonial videos are the highest-converting content type

### Channel Setup
1. Create: "Hyderabad School of Music" channel
2. Profile photo: HSM logo
3. Banner: Photo of classroom with tagline "Your first class is free"
4. Add website link + booking link in channel description

### Content Plan

| Series | Description | Frequency |
|---|---|---|
| Student Recitals | Upload every student performance from recitals | After each recital |
| Teacher Intros | 2-min video: "Hi, I'm [Name], I teach [Instrument] at HSM" | One per teacher |
| Learn [Instrument] in 30 Days | Beginner journey series — real student, real progress | 1 video/2 weeks |
| Parent Testimonials | "Why we chose HSM" — 2-3 minute interviews | 1 per month |
| Instrument Explainers | "What is Carnatic Vocals?" "How is Tabla tuned?" | 1 per month |

### YouTube SEO
- Title format: "[Instrument] Classes in Hyderabad | Student Performance | Hyderabad School of Music"
- Description: Include location, instruments, website link in first 3 lines
- Tags: "music classes Hyderabad", "[instrument] classes Hyderabad", "music school Kismatpur"
- Playlists: Organise by instrument and content type

## Content Production Workflow

### Minimum viable setup (phone camera is fine)
- **Equipment:** Any smartphone with decent camera, a ring light (₹1,500 on Amazon), a lapel mic (₹2,000)
- **Editing:** CapCut (free) or InShot — both have auto-captions which are essential for silent scroll
- **Storage:** One Google Drive folder per month, organised by type (Reels/Stories/Photos)

### Batch filming sessions (do this monthly)
- **30-minute session with 3–4 students:** Film 4–5 Reels in one go
- **15-minute session with each teacher:** Film teacher intro + 1 tip video
- This creates 3–4 weeks of content in one session

### Caption template

```
[Hook line — bold, curiosity-driving]

[2–3 lines of value or story]

[CTA — one of:]
• "Comment 🎸 if you want us to cover [instrument] next"
• "Book your free class → link in bio"
• "Tag a friend who should try this"

[Hashtags — 10–12]
```

## Monthly Review Checklist

At the end of each month, review:
- [ ] Top 3 performing Reels (by reach, not just likes) → create more like these
- [ ] Follower count vs. target — adjust if off track
- [ ] WhatsApp opt-ins from trial form — are leads coming in from social?
- [ ] Google reviews added this month (target: 5+/month)
- [ ] Which hashtags drove the most impressions — refine the set
- [ ] Any competitor content that performed well — analyse and adapt the format (never the content)
